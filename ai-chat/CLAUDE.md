# Claude Instructions

이 프로젝트는 클로드 입문 책을 따라하는, 완벽한 비개발자에 의해 진행되는 바이브 코딩 프로젝트이다.

이 프로젝트를 바이브코딩하는 비개발자는 개발 지식이 전무하며, "이것을 알 것이다" 라는 전제를 깔지 말고 쉽게, 그리고 자세히 설명하라.

이 프로젝트를 가이드하는 클로드 입문 책은, 개발자를 위해서 쓰여졌지만 이 책을 실습하는 비개발자는 이걸 모르고 잘못 구입한 사람이다.

그러므로 최대한 친절하게, 쉽게 설명하라.

---

# 프로젝트 명세

## 프로젝트 개요
영어 학습을 위한 AI 챗봇 웹 애플리케이션.
사용자가 영어로 자유롭게 대화하면 AI가 대화를 이어가면서 문법/표현 교정과 더 나은 표현을 제안해준다.
추후 모바일 앱으로 전환 가능한 구조로 개발한다.

## 핵심 기능
- 영어 자유 대화 연습
- 대화 중 실시간 문법/표현 교정 및 더 자연스러운 표현 제안
- 한국어/영어 모두 지원 (영어로 대화하다가 한국어로 질문 가능)
- 로그인 없이 바로 사용 가능
- 대화 기록은 세션(브라우저 탭) 중에만 유지, 세션당 메시지 수 무제한

## UI/UX
- 카카오톡 스타일의 말풍선 채팅 인터페이스
- 반응형 디자인 (PC, 태블릿, 모바일 모두 지원)
- 웹 브라우저에서 사용 (추후 모바일 앱 전환 가능한 구조)

## 기술 스택

### 프론트엔드 & 백엔드
- **프레임워크**: Next.js (App Router)
- **API 서버**: Hono
- **스타일링**: Tailwind CSS

### AI
- **모델**: gemini-2.0-flash (Google Gemini)
  - 이유: 무료 사용 가능 (하루 1,500회 요청), 성능 우수
  - API 키 발급: https://aistudio.google.com/apikey
- **에이전트 프레임워크**: Mastra

### 배포
- **플랫폼**: Google Cloud Run
- **예상 동시 접속**: 5~10명

## 개발 원칙
- 비개발자도 이해할 수 있도록 코드 변경 시 무슨 작업을 하는지 한국어로 설명한다
- 단계별로 나눠서 진행하고, 각 단계가 끝나면 확인을 받는다
- 오류가 발생하면 원인과 해결 방법을 쉬운 말로 설명한다
